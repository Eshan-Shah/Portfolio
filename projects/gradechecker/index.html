How It Works

The app takes in the users Exam Level, Exam Board and Marks then converts it into a percentage. Using the data found from the online grade boundaries, it creates a table calculating the percentage for each grade. It then compares the percentage achieved by the user with the table and returns all relevant information to the user so they can see what they achieved and how close they were to surrounding grades.
