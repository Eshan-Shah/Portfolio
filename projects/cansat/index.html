How It Works

The device uses an Arduino Nano Every as the microcontroller, this is paired with multiple sensors to record values of Temperature, Pressure, Humidity and more. Using a series of complex equations many numerical values are returned which can then be formatted into a database. A trend is formed via linear interpolation and regresssion calculations to then predict how the weather will follow the trend in the near future. The weather estimates can be accurate for up to 8 hours which is quite long considering the whole flight time is sub 1-minute.
